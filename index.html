Faster Snake Game

A faster-paced version of the classic snake game, created using the turtle module. This version of snake has an in-game timer that increases when the user (snake) eats a food object. Unlike the original game, additional food is also created every 10 seconds, and food is deleted from the screen around 10 seconds after it was created. The food changes color from blue -> yellow -> red as it gets close to being deleted. These unique features were added to the classic snake game to provide users with a fresh experience that requires fast thinking and dynamic gameplay.
