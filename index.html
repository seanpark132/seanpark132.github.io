Faster Snake Game

A faster-paced version of the classic snake game, created using python's turtle module. This version of snake has an in-game timer that increases when the user (snake) eats a food object. Unlike the original game, additional food is also created every 10 seconds, and food is deleted from the screen around 10 seconds after it was created. The food changes color from blue -> yellow -> red as it gets close to being deleted. These unique features were added to the classic snake game to provide users with a fresh experience that requires fast thinking and dynamic gameplay.

Python Syntax Flash Cards / Quiz GUI

A flash cards/quiz app built on Python using the tkinter Graphical User Interface (GUI) module. The app displays lines of python code, and the user must enter the correct output to the code within a time limit. After pressing the "FLIP" button, the correct output, the user's answer, and the definition of the function used are shown. The purpose of this app is to provide users with an interface where they can test their knowledge of a subject, such as Python syntax.

Cheap Flight Finder

A program that texts and emails the user with cheap flight deals for a list of destinations. For each requested flight destination, flight data for the lowest priced ticket in the next 6 months is pulled from Tequila's travel API. If the prices of these tickets are low compared to historical lows, the program will text the user with the flight's information and price. The program will also email users with a link to book the flight.

  • Github source code

Rain Alert Program

A program that checks the daily weather forecast and texts the user in the morning if it will rain at the location they input. Hourly weather data is pulled from OpenWeatherMap's API and the program checks if there will be any type of rainy weather in the next 12 hours. If the forecast has rainy weather, the program will text the user to bring an umbrella, by using the Twilio API.
